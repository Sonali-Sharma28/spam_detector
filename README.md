# 📧 Spam Detector — NLP Project

Classifies SMS messages as spam or normal using 
classical NLP and machine learning.

## Accuracy: ~97-98%

## NLP Pipeline
Raw text → Lowercase → Remove punctuation →
Remove stop words → Lemmatize → TF-IDF → Model

## Models compared
- Naive Bayes ← classic spam filter algorithm
- Logistic Regression
- Random Forest
